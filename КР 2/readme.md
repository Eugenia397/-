# Условие:
Решить задачу о рюкзаке (0-1) приближённо: отсортировать предметы по удельной стоимости и жадно набирать.
Алгоритм: жадная эвристика для рюкзака.

# Описание работы алгоритма:

**Алгоритм решает задачу о рюкзаке жадным методом:**

Создаются пары («удельная стоимость», «индекс») для каждого предмета (items[i].first = static_cast<double>(values[i]) / weights[i]).

Все предметы сортируются по убыванию удельной стоимости (sort(items.begin(), items.end(),...)).

Затем проходят по отсортированным предметам, выбирая их в порядке убывания выгоды, пока позволяет вместимость рюкзака (totalWeight + weights[item.second] <= W).

Наконец, возвращаются суммарная ценность выбранных предметов (return totalValue).

# Временная сложность:

Алгоритм состоит из двух основных частей:

Сортировка: О(n log n)

Мы используем стандартный метод сортировки, имеющий временную сложность O(n log n), где n — число предметов.

Проход по списку: O(n)

Далее перебор списка выполняется линейно, за O(n).

Таким образом, общая временная сложность составляет: O(nlogn)+O(n)=O(nlogn)

# Описание временной сложности:
Основной вклад в сложность даёт операция сортировки.
Хотя цикл прохода по данным занимает O(n), он значительно менее значимый, поскольку константа перед операцией сортировки намного больше.
Поэтому доминирующей частью является именно сортировка.

# Ответ на контрольный вопрос:

Жадный алгоритм для задачи покрытия множеств обеспечивает коэффициент аппроксимации H(d), где H(d) — гармоническое число порядка d.

Это значит, что найденное решение гарантированно не превышает оптимальное более чем в H(d) раз, где d — максимальное число множеств, покрывающих один элемент.
